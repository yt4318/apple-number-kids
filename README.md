# 🍎 Apple Number Kids

子供向けの数字あてゲームです。表示された数字と同じ数のりんごが描かれたカードを選びます。

## 遊び方

1. スタートボタンを押してゲーム開始
2. 1〜10の数字が大きく表示されます
3. 4つのカードにはりんごがピラミッド型で並んでいます
4. 数字と同じ個数のりんごのカードをタップ
5. 正解すると ○（ピンポン音）、不正解だと ×（ブー音）が表示されます
